# Typescript Notes

## Installing Typescript

1. Install Typescript

```bash
npm init -y
npm install typescript --save-dev

```

2. Create a tsconfig.json File

```bash
npx tsc --init

```

3.  Compile the TypeScript Code

```bash
npx tsc index.ts
```

4. Compile the TypeScript Code => watch mode

```bash
npx tsc --watch

```
